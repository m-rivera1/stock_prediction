Project Description

Welcome to Stock Predictions!

Warren Buffet said it best when he said, "If you aren’t thinking about owning a stock for ten years, don’t even think about owning it for ten minutes.” Using Machine Learning, you will be able to use the Stock Prediction tool to help you decide which stocks to invest in. The Stock Prediction model operates by following a stock's previous patterns and predicting a new pattern for the next seven days. Then, you will be able to note the accuracy of each 'trained' prediction pattern.
